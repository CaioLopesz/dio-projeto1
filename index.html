O que são as TAGS?

(voltar)

As TAGS no HTML são como marcadores que dizem ao navegador da web como mostrar o conteúdo de uma página web.

Por exemplo, usamos TAGS para dizer onde começa e termina um parágrafo, onde está uma imagem, onde está um link para clicar entre outras funções. As TAGS são como instruções que dizem ao computador como organizar e mostrar tudo o que está na página.

Como as TAGS são construídas

(voltar)

Uma TAG é construída por 3 partes:

  1. Uma TAG de abertura, começando com os símbolos '< >'
  2. O conteúdo, que fica entre a TAG de abertura e entre a TAG de fechamento. As informações colocadas entre elas aparecerão para o usuário
  3. Uma TAG de fechamento, terminando com os símbolos ''

TAGS de formatação de texto

(voltar)

As TAGS voltadas para a formatação de texto é uma ferramenta muito importante se queremos deixar o nosso texto mais fácil para a leitura. Algumas TAGS de formatação são:

  • tags h1 até h6: Essas TAGS são usadas para criar cabeçalhos de diferentes tamanhos. O h1 é o maior e o h6 o menor. Eles são usados para destacar seções de textos, como títulos e subtítulos.
  • tags 'p': Essa TAG é usada para criar parágrafos de texto. Ela é usada sempre que queremos começar um novo parágrafo em uma página da web.
  • tag 'strong': Essa TAG é usada para deixar o texto em negrito.
  • tag 'i': Essa TAG é usada para tornar o texto em itálico.
  • tag 'u': Essa TAG é usada para sublinhar o texto. Ela é usada quando queremos destacar ou enfatizar uma parte do texto com um sublinhado.

TAGS de lista

(voltar)

Basicamente, TAGS de lista ajudam a organizar informações de maneira fácil e clara de ler em uma página web, sendo bem simples de usar.

As TAGS de lista são:

  • tag 'ul': Essa TAG é usada para criar uma lista não ordenada, onde os itens não têm uma ordem específica.
  • tag 'ol': Essa TAG é usada para criar uma lista ordenada, onde os itens são numerados ou têm outro tipo de ordem específica.
  • tag 'li': Essa TAG é usada para definir cada item dentro de uma lista, seja ele em uma lista ordenada ou não ordenada. Ou seja, utilizamos a TAG 'li' dentro da TAG 'ul' ou da TAG 'ol'.

TAGS de link

(voltar)

Uma TAG de link é usada para criar um vínculo entre uma página da web e outra, ou entre diferentes partes da mesma página.

Ela é representada pela TAG 'a'.

Tributos de TAGS

(voltar)

Os tributos de TAGS são como características especiais que podemos adicionar às TAGS para fazer certas coisas acontecerem. Eles fornecem informações extras sobre como queremos que uma TAG funcione ou apareça na página da web.

Por exemplo, se quisermos adicionar uma imagem a uma página, usamos a TAG 'img', e podemos incluir um atributo chamado 'src' para dizer ao navegador onde encontrar a imagem que queremos exibir.

Outro exemplo é quando usamos a TAG 'a' para criar um link. Podemos usar o atributo 'href' para indicar qual página ou recurso queremos vincular.

Os atributos podem variar de acordo com a TAG que estamos usando e podem fazer coisas diferentes, como alterar cores, tamanhos, posições e muito mais. Resumindo, os atributos são como instruções extras que damos às TAGS para personalizar o comportamento ou a aparência dos elementos em nossas páginas da web.
